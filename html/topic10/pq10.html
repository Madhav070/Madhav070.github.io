Mini Project

Suppose ABC is a private limited company which manages the employee records of other companies. Employee id can be of any length, and it can contain any character. The following are the task you have to perform for three employees.
1.Take the length of an employee id as an input and store it in an integer length variable
2.Take an employee id as an input and display it on the screen.
3.Save the employee id in a character array, which is allocated dynamically.
4.Create only one character array dynamically.
